Centeva Interview Question

For this task you will be given an initial set of source code which contains an Angular application that is served up by a local NPM server through a grunt task. You will take this source code and complete the user story described below.

Setup

  1. Create a local git clone of the repo.
  2. Ensure that you have NPM/Node installed.
  3. Setup the project by running the following commands from the command prompt at the root of the repo
npm install -g yo
npm install -g grunt-cli
npm install -g generator-centeva-angular
npm install -g bower
npm install
bower install

Tips

  • Take a moment and familiarize yourself with the packages included in the source, including checking out their respective documentations. Examples include, bootstrap, angular, moment, etc.
  • Check in frequently. Don't worry about mistakes in prior commits.

User Story

The user needs a system for managing milestones on a project, to do this we are going to create a list of project milestones.

For now we are only worrying about add and view, no edit or delete.

Acceptance Criteria

  • Clicking the add milestone button opens a dialog (see mockup) that allows the user to specify a name and a date for the milestone.
  • Saved milestones are added to the main milestone list page.
  • Milestones should be sortable by date or name.
